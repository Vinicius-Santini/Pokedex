# POKEDEX (React)


## O que é necessário para rodar:

 - Apenas utilizar o comando: ```npm i``` para instalação das dependências.
 
 
 ## Descrição:
 O objetivo deste projeto é consumir a [PokeAPI](https://pokeapi.co/) utilizando **React.js**.
